# 🛒 Blinkit Sales Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the Blinkit Grocery Sales dataset. The dashboard provides valuable insights into sales performance, customer preferences, outlet performance, and product trends, enabling data-driven business decisions.

---

## 🎯 Business Objective

The objective of this project is to analyze Blinkit's sales data and identify key business insights related to product performance, outlet efficiency, customer ratings, and overall sales trends.

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel

---


## 📈 Key Performance Indicators (KPIs)

- 💰 Total Sales
- 📦 Total Items Sold
- 💵 Average Sales
- ⭐ Average Rating

---

## 📊 Dashboard Features

- Sales by Outlet Location
- Sales by Outlet Size
- Sales by Outlet Type
- Sales by Item Type
- Fat Content Analysis
- Outlet Establishment Trend
- Interactive Filter Panel

---

## 💡 Key Business Insights

- Medium-sized outlets generated the highest sales.
- Fruits & Vegetables contributed the highest sales among product categories.
- City 3 achieved the highest revenue.
- Sales peaked for outlets established in 2018.
- Customer ratings remained consistent across different outlet types.
